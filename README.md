# Чистота

Учебный проект по теме клининговой компании.

## Запуск

```powershell
npm start
```

Сайт откроется по адресу:

```text
http://localhost:3000
```

## Демо-входы

Администратор:

```text
admin@chistota.ru
admin123
```

Пользователь:

```text
elena@mail.ru
user123
```

## База данных

SQLite-файл создаётся автоматически:

```text
data/database.sqlite
```
